Use Cases

The primary purpose of the site is to serve as additional information for a curriculum vitae (C.V.) and all users recognized that. Other intended uses were to serve as an information page for patrons and faculty, but that use is not met by the current design. Support for those users will likely be abandoned as they would require a completely different structure and information set. Additionally, the page was intended to serve as information for students and/or future job seekers to demonstrate the skills and knowledge of someone currently holding a position, but again, that use was not recognized, even by the evaluator who is a current graduate student and future job-seeker.

Recommended / Planned changes to the design

The following changes were recommended and will be adopted to improve usability and aesthetics:

  • One spelling error was noticed, as well as having all page titles the same. Those corrections will be easily made.
  • More comprehensive contact information will be added and the footer updated to increase font size as well as reflect the updated date, rather than the created date.
  • Software packages will be linked out to their manufacturer's websites as well as having brief descriptions of what the software does and/or how it is used in a library and/or academic setting.
  • Some effort must be made to improve navigation and images added to break up lengthy text blocks.

Reflection on evaluation process

It was useful to have additional sets of eyes to evaluate the site, not only for content, but for coding purposes. Co-workers familiar with the library environment and hiring process were useful for evaluating general content, but it was also beneficial to have other read through and evalate. Non-library users were able to pay closer attention to style details and not be so consumed with the actual content. I particularly appreciated the input of my friend who is a former web-developer; he was able to look closely at the html and css and ask intelligent questions about the structure of the site that should help with improving the overall feel and efficiency of the site.
